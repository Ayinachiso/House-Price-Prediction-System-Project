About the Model

This system uses a Linear Regression model trained on real house sales data. It analyzes six key features to predict the likely sale price of a home in Ames, Iowa.

Features Used:
  • Overall Quality
  • Living Area (sq ft)
  • Basement Area (sq ft)
  • Garage Capacity
  • Bedrooms Above Ground
  • Neighborhood

How to Use

  • Enter realistic values for each feature.
  • Neighborhood is case-sensitive (e.g. CollgCr).
  • Click Predict Price to see the estimated value.
  • Try different values to see how features affect price!
